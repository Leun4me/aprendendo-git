# aprendendo-git 

## Comandos básicos do git

```sh
git init #para iniciar um repositório
git remote add origin link_do_repositorio_do_github # pegar o link no github
git add . # adiciona todos os arquivos para serem monitorados
git commit -m "uma mensagem entre aspas"
git push origin main # envio de dados para o github
git pull origin main # recebe arquivos modificados do github # mostra se tem arquivos criados, modificados ou excluídos.
#também informa se o repo remoto está atualizado com o local e vice versa git status

git status
```
